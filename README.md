## Selected Projects
- **ORION – WebAssembly Compiler**  
  A compiler written in TypeScript that compiles a custom low-level language directly into raw WebAssembly binary modules without relying on existing toolchains or WAT.  
  https://github.com/vennictus/orion-wasm-compiler

- **benz – Graph Query Execution Engine**  
  An in-memory graph query engine implementing a lazy, pull-based Volcano-style execution model to demonstrate how modern traversal pipelines work internally.  
  https://github.com/vennictus/benz-graph-engine

- **kilo – Terminal Text Editor in C**  
  A lightweight terminal text editor written in pure C using raw terminal control, ANSI escape sequences, and low-level file operations, with zero external libraries.  
  https://github.com/vennictus/kilo-txt-editor
